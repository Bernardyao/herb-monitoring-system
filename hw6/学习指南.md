# 中药种植监控系统 - 学习指南

## 1. 项目概述
本项目旨在实现一个中药种植监控系统，通过传感器监测环境参数（如温度和湿度），并提供相应的用户界面进行数据展示和管理。

## 2. 环境准备
- **Java JDK**: 确保安装了Java JDK 8或更高版本。
- **Maven**: 安装Maven以便于项目的构建和依赖管理。
- **IDE**: 推荐使用IntelliJ IDEA或Eclipse进行开发。

## 3. 项目结构
项目采用分层架构，主要分为以下几个部分：
- **dao**: 数据访问层，负责与数据库的交互。
- **model**: 数据模型，定义了传感器及其数据的实体。
- **sensor**: 传感器相关的实现，包括传感器的生成和数据处理。
- **service**: 业务逻辑层，处理传感器的操作和阈值监控。
- **ui**: 用户界面层，提供用户交互的窗口和组件。

## 4. 编译与运行
### 编译项目
在项目根目录下运行以下命令以编译项目：
```bash
mvn clean install
```

### 运行项目
编译完成后，使用以下命令运行项目：
```bash
mvn exec:java -Dexec.mainClass="com.herbs.monitoring.AppLauncher"
```

## 5. 功能介绍
- **登录功能**: 用户通过登录界面输入凭据（默认：admin/admin）进入主界面。
- **数据监控**: 主窗口展示传感器数据，并支持CRUD操作。
- **阈值设置**: 用户可以通过配置文件设置温度和湿度的告警阈值。

## 6. 学习建议
- **代码阅读**: 深入阅读源代码，理解各个模块的实现。
- **功能扩展**: 尝试添加新的传感器类型或改进现有功能。
- **UI优化**: 研究Swing组件，优化用户界面布局和交互体验。

## 7. 常见问题
- **编译错误**: 确保Maven依赖已正确下载，检查`pom.xml`文件。
- **运行时异常**: 检查数据库连接配置，确保SQLite数据库文件存在。
